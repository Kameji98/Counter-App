# Counter App (Vanilla JS)

A minimal counter app with increment, decrement, and reset controls.

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Run Locally
Open `src/index.html` in your browser.

## Project Structure
- `src/index.html` — UI
- `src/styles.css` — styling
- `src/app.js` — logic

## What I practiced
- DOM events and state updates
- Simple utility functions (clamp)
